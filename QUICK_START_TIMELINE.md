# 🚀 分镜时间线编辑 - 快速开始

## ⚡ 60秒快速入门

### 打开时间线编辑器

**方式1**: 从项目仪表板
```
1. 打开 dashboard.html
2. 找到"午后的相遇"项目卡片
3. 点击 📊 时间线 按钮
```

**方式2**: 从工作台
```
1. 打开 index.html
2. 左侧侧边栏找到 📊 分镜时间线
3. 点击进入
```

### 第一次使用 - 创建一个新分镜

1. **打开时间线编辑器** (选择上面任一方式)

2. **填写右下方的表单**:
   ```
   分镜名称: 输入你的分镜标题
   分镜顺序: 输入数字 (6, 7, 8...)
   叙事阶段: 从下拉菜单选择一个阶段
   简介: 描述这个分镜的内容
   生成状态: 选择"待生成"或"已生成"
   ```

3. **点击 "创建分镜" 按钮**

4. **完成!** ✓
   - 新分镜会立即出现在左下方的分镜库
   - 返回工作台时会看到这个新分镜

## 📖 5种主要操作

### 1️⃣ 编辑阶段名称

```
左侧面板 → 找到要修改的阶段 → 点击 ✏️ 按钮 → 输入新名称 → 确定
```

**例如**: 把"铺垫"改成"开篇"

### 2️⃣ 重新排序阶段

```
左侧面板 → 按住 ⋮⋮ 图标 → 拖拽到新位置 → 释放
```

**结果**: 阶段顺序改变，自动保存

### 3️⃣ 查看时间线分布

```
中上方看到5列网格，每列显示一个阶段
```

**显示内容**:
- 阶段顺序号
- 阶段名称
- 分配到该阶段的分镜数量

### 4️⃣ 创建新分镜

```
右下方表单 → 填写信息 → 点击"创建分镜"
```

**包含的信息**:
- 分镜名称
- 分镜顺序
- 所属阶段
- 简介
- 生成状态

### 5️⃣ 返回工作台

```
点击顶部 "返回工作台" 按钮 → 回到 index.html
```

**已创建的分镜会在工作台显示**

## 💾 数据说明

### 您的数据存储在哪里?
- **浏览器本地存储** (localStorage)
- **自动保存**: 你创建或编辑的任何内容都会立即保存
- **无需云端**: 所有操作都是本地的

### 数据会消失吗?
- ✅ **不会** - 直到你清除浏览器数据
- ✅ **跨设备同步** - 在工作台编辑的内容会在时间线显示
- ✅ **刷新保存** - 即使刷新页面也不会丢失

### 如何备份?
1. 打开浏览器开发者工具 (F12)
2. 找到 Application → LocalStorage
3. 导出 `project_data` 的值

## ❓ 常见问题

**Q: 创建的分镜在哪里看?**  
A: 
- 在时间线编辑器的左下方"分镜库"中
- 返回工作台会看到它在分镜列表中

**Q: 编辑后的数据会保存吗?**  
A: 是的，自动保存到浏览器本地存储

**Q: 能删除阶段吗?**  
A: 暂不支持删除，但可以编辑名称

**Q: 创建的分镜太多怎么办?**  
A: 都会显示在分镜库中，可以在工作台中管理

**Q: 分镜顺序错了怎么办?**  
A: 
1. 返回工作台
2. 在分镜控制中编辑
3. 或者创建一个新分镜用正确的顺序

## 💡 最佳实践

### ✅ 推荐做法
1. 定期创建分镜，保持故事节奏
2. 使用有意义的分镜名称
3. 为不同阶段分配适当数量的分镜
4. 添加详细简介帮助回忆内容

### ❌ 避免做法
1. 过多地修改阶段名称（容易混淆）
2. 创建分镜时顺序重复
3. 忘记返回工作台编辑详情
4. 在分镜库中进行大量操作
