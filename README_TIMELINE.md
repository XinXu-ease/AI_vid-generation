## ❓ 常见问题快速解答

**Q: 创建的分镜在哪里看?**
A: 在分镜库（左下）和工作台中都能看到

**Q: 数据会丢失吗?**
A: 不会，存储在本地，除非清除浏览器数据

**Q: 能删除分镜吗?**
A: 当前不支持删除，但可以在工作台中隐藏

A: 支持从 localStorage 导出，见故障排查

A: 暂不支持，已列为未来功能
